HGE — Hardware accelerated 2D games engine

This is contributed work based on HGE 1.8.1 configured with CMake (DirectX8 and DirectX9 version in the same source tree), with shaders support, fix for window borders on Vista/7, and reformatted source code.

Haaf's Game Engine (HGE)

Welcome to GitHub Pages for HGE, a 2D graphic engine for Windows utilizing DirectX8 and DirectX9 (flavours for Unix and OpenGL also exist, made by third parties— seehge-unix).

Read Documentation HERE

Authors and Contributors

This repository on Github is managed by Dmytro Lytovchenko (@kvakvs) and contains original HGE plus CMake build scripts and few patches to add shaders and fix few problems. Dmytro is NOT Haaf, who is the original author of HGE.
